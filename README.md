# Hrishikesh - AI & ML Engineer Portfolio

## Overview
This is a highly interactive, beautifully designed single-page portfolio website tailored for Hrishikesh, an AI & Machine Learning Engineer. It features a modern, futuristic aesthetic with a dark theme, neon accents, and smooth 3D animations.

## Features
- **Immersive Loader**: A custom animated loader featuring a tech-guy SVG avatar, scanline effects, and a dynamic progress bar.
- **Hero Section**: A stunning entrance with 3D text reveals, floating tech badges, and parallax effects.
- **About Section**: Details about Hrishikesh, accompanied by an interactive 3D spinning cube showcasing different tech stacks, and a glowing skill cloud.
- **Projects Showcase**: Interactive project cards with magnetic hover effects, unique gradient backgrounds, and responsive layouts.
- **Experience Timeline**: A clean, chronological journey of professional milestones with animated indicators.
- **Tech Stack Grid**: A comprehensive display of technical proficiencies with slick hover animations.
- **Custom Cursor & Smooth Scrolling**: Enhances the overall premium feel of the user experience.

## Technology Stack
- **HTML5**: Semantic structure.
- **CSS3 (Vanilla)**: Advanced styling, CSS variables for theming, complex keyframe animations, 3D transforms, and responsive design.
- **JavaScript (Vanilla)**: (Assumed built-in or linked logic for cursor tracking, scroll reveals, and interactivity).

## Deployment
This project is configured to be easily deployed to static hosting platforms such as Netlify, Vercel, or GitHub Pages.

## Usage
Simply open `index.html` in any modern web browser to view the portfolio. No build step or build tools are required for the front-end code.
